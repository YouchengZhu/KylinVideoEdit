# KylinVideoEdit
用C++、Qt、FFmpeg实现的视频剪辑器，可剪辑音频、视频，完成基本的编辑操作。

## 功能
* 音频、视频的播放，倍速等播放控制；
* 音视频的拆分、裁剪、合并等操作；
* 为视频添加背景音乐；
* 支持多种音视频格式的转换；
* 提供截图、添加画中画等附加功能；

## 环境要求
* Linux
* C++11
* Qt5.15.2
* FFmpeg
